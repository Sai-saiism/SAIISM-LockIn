## 🌐 Official Website

## 🎯 Project Overview
SAIISM-LockIn is a proprietary, minimalist web application designed for strict, distraction-free 1-to-1 communication. In a world of noise, we're building a quiet room for two. Hosted securely on Netlify.

## ✨ Key Features
* **Strict 1-to-1 Lock System:** Users can only connect and chat with one partner at a time. No groups, no external search once locked.
* **Destructive Exit & Security PIN:** Exiting a chat requires a 4-digit security PIN, which instantly and permanently deletes all chat history from the Firestore database.
* **Minimalist UI/UX:** A clean, 90% Black & White interface to ensure maximum focus on communication.
* **Client-Side Media Compression:** Images are compressed into Base64 strings on the client side before transmission to ensure speed and minimize server load.

## 🔒 Privacy & Architecture
This project prioritizes user consent and data control. It utilizes Firebase Authentication (Google Sign-in) and Firestore for real-time synchronization.
